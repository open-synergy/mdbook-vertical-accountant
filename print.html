Membuat Accountant Service

A. INPUT

(Tidak ada instruksi khusus)

B. LANGKAH KERJA

  1. Buka menu Accountant Report -> Configuration -> Accountant Service. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Klik tombol Create pada bagian atas-kiri form.

  1. Isi Name. Wajib diisi.
  2. Isi Internal Reference. Tidak wajib diisi.
  3. Aktifkan/ Deaktifkan Active. Tidak wajib diisi.
  4. Aktifkan/ Deaktifkan Assurance Service. Tidak wajib diisi.
  5. Isi Code. Tidak wajib diisi.
  6. Pilih Sequence Creation Method. Wajib diisi.
  7. Beralih ke tab Python Code.
  8. Isi Python Code for Custom Sequence Generation. Tidak wajib diisi.
  9. Beralih ke tab Signing Partner.
  10. Tambah/Modifikasi/Hapus Signing Partner. Ulangi langkah ini sampai Signing Partner sesuai dengan keinginan.
  11. Beralih ke tab Opinions.
  12. Aktifkan/ Deaktifkan Opinion Required. Tidak wajib diisi.
  13. Tambah/Hapus Opinion. Ulangi langkah ini sampai Opinion sesuai dengan keinginan.
  14. Beralih ke tab Quality Control.
  15. Tambah/Modifikasi/Hapus Question. Ulangi langkah ini sampai Question sesuai dengan keinginan.
  16. Beralih ke tab Methods.
  17. Aktifkan/ Deaktifkan Method Required. Tidak wajib diisi.
  18. Tambah/Hapus Method. Ulangi langkah ini sampai Method sesuai dengan keinginan.
  19. Beralih ke tab Policy.
  20. Pilih Allowed To Confirm. Tidak wajib diisi.
  21. Pilih Allowed To Validate. Tidak wajib diisi.
  22. Pilih Allowed To Cancel. Tidak wajib diisi.
  23. Pilih Allowed To Restart. Tidak wajib diisi.
  24. Isi Description. Tidak wajib diisi.
  25. Klik tombol Save pada bagian atas-kiri form.

C. OUTPUT

  • Data Accountant Service akan terbuat.

  5. Isi dan sesuaikan Code jika dibutuhkan. Tidak wajib diisi.
  6. Pilih dan sesuaikan Sequence Creation Method jika dibutuhkan. Wajib diisi.
  7. Beralih ke tab Python Code.
  8. Isi dan sesuaikan Python Code for Custom Sequence Generation jika dibutuhkan. Tidak wajib diisi.
  9. Beralih ke tab Signing Partner.
  10. Tambah/Modifikasi/Hapus Signing Partner. Ulangi langkah ini sampai Signing Partner sesuai dengan keinginan.
  11. Beralih ke tab Opinions.
  12. Aktifkan/ Deaktifkan Opinion Required jika dibutuhkan. Tidak wajib diisi.
  13. Tambah/Hapus Opinion. Ulangi langkah ini sampai Opinion sesuai dengan keinginan.
  14. Beralih ke tab Quality Control.
  15. Tambah/Modifikasi/Hapus Question. Ulangi langkah ini sampai Question sesuai dengan keinginan.
  16. Beralih ke tab Methods.
  17. Aktifkan/ Deaktifkan Method Required jika dibutuhkan. Tidak wajib diisi.
  18. Tambah/Hapus Method. Ulangi langkah ini sampai Method sesuai dengan keinginan.
  19. Beralih ke tab Policy.
  20. Pilih dan sesuaikan Allowed To Confirm jika dibutuhkan. Tidak wajib diisi.
  21. Pilih dan sesuaikan Allowed To Validate jika dibutuhkan. Tidak wajib diisi.
  22. Pilih dan sesuaikan Allowed To Cancel jika dibutuhkan. Tidak wajib diisi.
  23. Pilih dan sesuaikan Allowed To Restart jika dibutuhkan. Tidak wajib diisi.
  24. Isi dan sesuaikan Description jika dibutuhkan. Tidak wajib diisi.

Membuat Partner Arrangement

A. INPUT

(Tidak ada instruksi khusus)

B. LANGKAH KERJA

  1. Buka menu Accountant Report -> Configuration -> Partner Arrangement. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Klik tombol Create pada bagian atas-kiri form.

  1. Ubah # Partner Arrangement dengan penomeran yang dikehendaki. Biarkan berisi / apabila menghendaki penomeran otomatis.
  2. Pilih Managing Partner. Wajib diisi.
  3. Pilih Arrangement Date. Wajib diisi.
  4. Beralih ke tab Partner.
  5. Tambah/Hapus Partner. Ulangi langkah ini sampai Partner sesuai dengan keinginan.
  6. Beralih ke tab Note.
  7. Isi Note. Tidak wajib diisi.
  8. Beralih ke tab Policy.
  9. Aktifkan/ Deaktifkan Can Confirm. Tidak wajib diisi.
  10. Aktifkan/ Deaktifkan Can Validate. Tidak wajib diisi.
  11. Aktifkan/ Deaktifkan Can Cancel. Tidak wajib diisi.
  12. Aktifkan/ Deaktifkan Can Restart. Tidak wajib diisi.
  13. Klik tombol Save pada bagian atas-kiri form.

C. OUTPUT

  • Data Partner Arrangement akan terbuat.

Memodifikasi Partner Arrangement

A. INPUT

  • Data Partner Arrangement yang dapat dimodifikasi harus memiliki status Draft.

  • User yang akan memodifikasi harus memiliki akses untuk memodifikasi Partner Arrangement.

B. LANGKAH KERJA

  1. Buka menu Accountant Report -> Configuration -> Partner Arrangement. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data Partner Arrangement yang akan dimodifikasi. Abaikan jika data sudah dibuka.
  3. Klik tombol Edit pada bagian atas-kiri form.

  1. Ubah # Partner Arrangement dengan penomeran yang dikehendaki. Biarkan berisi / apabila menghendaki penomeran otomatis.
  2. Pilih dan sesuaikan Managing Partner jika dibutuhkan. Wajib diisi.
  3. Pilih dan sesuaikan Arrangement Date jika dibutuhkan. Wajib diisi.
  4. Beralih ke tab Partner.
  5. Tambah/Hapus Partner. Ulangi langkah ini sampai Partner sesuai dengan keinginan.
  6. Beralih ke tab Note.
  7. Isi dan sesuaikan Note jika dibutuhkan. Tidak wajib diisi.
  8. Beralih ke tab Policy.
  9. Aktifkan/ Deaktifkan Can Confirm jika dibutuhkan. Tidak wajib diisi.
  10. Aktifkan/ Deaktifkan Can Validate jika dibutuhkan. Tidak wajib diisi.
  11. Aktifkan/ Deaktifkan Can Cancel jika dibutuhkan. Tidak wajib diisi.
  12. Aktifkan/ Deaktifkan Can Restart jika dibutuhkan. Tidak wajib diisi.
  13. Klik tombol Save pada bagian atas-kiri form.

C. OUTPUT

  • Data Partner Arrangement akan berubah sesuai dengan perubahan yang dilakukan.
