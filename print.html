Assurance Report

Assurance Report adalah fitur yang digunakan untuk transaksi assurance report.

  • Penjelasan Assurance Report
  • Membuat Assurance Report
  • Memodifikasi Assurance Report
  • Menghapus Assurance Report
  • Mengkonfirmasi Assurance Report
  • Memvalidasi Assurance Report
  • Memilih Jawaban Item Pertanyaan Qualitative
  • Mengisi Jawaban Item Pertanyaan Quantitative
  • Membatalkan Assurance Report
  • Merestart Assurance Report

Penjelasan Assurance Report

Informasi pada Assurance Report dibagi menjadi beberapa bagian, yaitu:

  • Header
  • Tab Client Informations
  • Tab Quality Control
  • Tab Note
  • Tab Policy
  • Tab QR Code

HEADER

Customer

Main Report Attachment

A. INPUT

(Tidak ada instruksi khusus)

  11. Pilih Date. Wajib diisi.
  12. Pilih Signing Accountant. Wajib diisi.
  13. Pilih Opinion. Wajib/tidak diisi tergantung dari accountant service yang dipilih.
  14. Pilih Main Report Attachment. Tidak wajib diisi.
  15. Beralih ke tab Client Informations.
  16. Pilih Information Based On. Tidak wajib diisi.
  17. Pilih Client Currency. Tidak wajib diisi.
  18. Isi Revenue. Tidak wajib diisi.
  19. Isi Total Asset. Tidak wajib diisi.
  20. Isi Total Liability. Tidak wajib diisi.
  21. Isi EBIT. Tidak wajib diisi.
  22. Isi Tax Expense. Tidak wajib diisi.
  23. Isi Total Net Profit. Tidak wajib diisi.

Menghapus Assurance Report

A. INPUT

  • Data Assurance Report yang dapat dihapus harus memiliki status Draft.

  • User yang akan menghapus harus memiliki akses untuk menghapus Assurance Report.

B. LANGKAH KERJA

  1. Buka menu Accountant Report -> Accountant Report -> Assurance. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data Assurance Report yang akan dihapus. Abaikan jika data sudah dibuka.
  3. Klik tombol More pada bagian atas-tengah form.

  1. Klik tombol Delete pada dropdown yang muncul ketika tombol More diklik.

  1. Klik tombol Ok pada pop-up konfirmasi penghapusan yang muncul.

C. OUTPUT

  • Data Assurance Report akan terhapus.

Mengkonfirmasi Assurance Report

A. INPUT

  • Data Assurance Report yang dapat dikonfirmasi harus memiliki status Draft.

  • User yang akan mengkonfirmasi harus memiliki akses untuk mengkonfirmasi Assurance Report.

B. LANGKAH KERJA

  1. Buka menu Accountant Report -> Accountant Report -> Assurance. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data Assurance Report yang akan dikonfirmasi. Abaikan jika data sudah dibuka.
  3. Klik tombol Confirm pada bagian atas-kiri form.

C. OUTPUT

  • Status dari Assurance Report akan berubah menjadi Waiting for Approval.

  • Isian Assurance Report sudah tidak bisa diubah.

Melakukan Finalisasi Assurance Report

  • Data Assurance Report yang akan dilakukan finalisasi harus memiliki status Waiting For Approval.

  • User yang akan melakukan finalisasi harus memiliki akses untuk melakukan finalisasi Assurance Report.

B. INSTRUKSI KERJA

  1. Buka menu Accountant Report -> Accountant Report -> Assurance. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data Assurance Report yang akan dilakukan finalisasi. Abaikan jika data sudah dibuka.
  3. Beralih ke tab Quality Control
  4. Klik tombol Edit pada bagian atas-kiri form.

  1. Jawab pertanyaan tiap data item Question pada tabel Quality Control. Pilih jawaban apabila isian Type == Qualitative atau Isi jawaban jika isian Type == Quantitative. Ulangi sampai semua jawaban sesuai dengan keadaan.
  2. Klik tombol Save pada bagian atas-kiri form.

  1. Klik tombol Finalization pada bagian atas-kiri form.

C. OUTPUT

  • Status Assurance Report akan berubah menjadi Finalization jika seluruh jawaban atas item pertanyaan Success? telah sesuai.

  • Data Assurance Report yang akan divalidasi harus memiliki status Waiting For Approval.

B. INSTRUKSI KERJA

  1. Buka menu Accountant Report -> Accountant Report -> Assurance. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data Assurance Report yang akan divalidasi. Abaikan jika data sudah dibuka.
  3. Beralih ke tab Quality Control
  4. Klik tombol Edit pada bagian atas-kiri form.

  1. Jawab pertanyaan tiap data item Question pada tabel Quality Control. Pilih jawaban apabila isian Type == Qualitative atau Isi jawaban jika isian Type == Quantitative. Ulangi sampai semua jawaban sesuai dengan keadaan.
  2. Klik tombol Save pada bagian atas-kiri form.

  1. Klik tombol Valid pada bagian atas-kiri form.
